# Cida Fialho - Depilação com Cera | Landing Page

Uma landing page institucional de alta conversão desenvolvida exclusivamente para o estúdio **Cida Fialho - Depilação com Cera**. O projeto foi planejado com foco na experiência do usuário (UI/UX), alta performance e total responsividade para atrair, conscientizar e converter novas clientes.

---

##  Sobre o Projeto

O objetivo principal deste site é consolidar a presença digital da profissional Cida Fialho, destacando seu método humanizado, biossegurança e experiência de mais de 15 anos no mercado de estética. 

A estrutura foi desenhada estrategicamente para guiar a visitante por uma jornada de valor:
1. **Hero Section:** Impacto visual imediato com quebra de objeções ("Você não precisa escolher entre uma pele lisa e uma pele saudável").
2. **Diferenciais & Benefícios:** Abordagem voltada para o acolhimento, biossegurança e mitigação de dores comuns (como allergies e foliculite).
3. **Serviços:** Apresentação clara e elegante dividida em três pilares essenciais: Faciais, Corporais e Íntima.
4. **Prova Social:** Carrossel dinâmico de depoimentos reais para gerar conexão e autoridade.
5. **Chamada para Ação (CTA):** Botões estratégicos direcionando para o agendamento direto pelo WhatsApp.

---

##  Design & Identidade Visual

O layout foi construído seguindo uma paleta de cores sofisticada e clean, que transmite calma, higiene e bem-estar:
* **Verde Sálvia:** Tonalidade principal, evocando naturalidade, saúde e relaxamento.
* **Tons de Creme/Off-White:** Fundo suave para garantir uma leitura leve e confortável.
* **Dourado/Mostarda:** Utilizado cirurgicamente em botões de ação e divisores para guiar o olhar da usuária aos pontos de conversão.
* **Tipografia:** Combinação elegante usando fontes Serifadas (`Playfair Display` / `Cormorant Garamond`) para títulos importantes, transmitindo prêmio e autoridade, combinadas com `Montserrat` / `Inter` para textos corridos focando em legibilidade.

---

##  Tecnologias Utilizadas

Para garantir o máximo desempenho, controle total de estilização e evitar o overhead de plugins pesados, o site foi desenvolvido utilizando **código puro (Vanilla CSS/JS)**:

* **HTML5:** Estrutura semântica e acessível.
* **CSS3:** Layouts modernos com CSS Grid e Flexbox, animações fluidas de entrada (`@keyframes`) e total adaptabilidade via Media Queries.
* **JavaScript (Vanilla):** Lógica nativa para controle de componentes interativos, como o carrossel dinâmico de depoimentos, sem dependência de bibliotecas externas (como jQuery).

---

##  Responsividade e Otimização

O site foi milimetricamente testado e otimizado para múltiplos breakpoints, garantindo uma experiência perfeita em:
*  **Desktop:** Layout amplo, com distribuição harmônica entre blocos de texto e imagens flutuantes.
*  **Tablets (iPad):** Grid inteligente que reajusta o tamanho dos cards de serviços e o comportamento do carrossel para exibição perfeita de um elemento por vez, evitando quebras visuais.
*  **Mobile (Smartphones):** Reordenação de elementos via Flexbox (`order`) para que imagens institucionais fiquem no topo antes do texto explicativo, respeitando a leitura vertical do celular.
